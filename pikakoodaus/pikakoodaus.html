5 Programming languages

Javascript

First appeared in 1995. Used for front-end web developement. Used to create dynamically updating content, control multimedia, animate images and many other things.

Python

First appeared in 1991. It is used as a support language for software developers, for build control and management, testing, and in many other ways.

C#

First appeared in 2000. It can be used for various applications, from desktop to web to mobile development. Ideal for building cloud-based apps.

C++

First appeared in 1985. The major advantage of C++ is performance. C++ performs efficiently and the speed is faster when compared to Python. C++ is suitable for almost every platform including embedded systems whereas Python can be used only on certain platforms that support high-level languages.

Rust

First appeared in 2015. Rust programming language is similar to C++ but it provides memory safety without using garbage collection. It is aimed at achieving higher performance and better safety than the C++ programming language.
